# Java Conditionals

This repository contains beginner-level Java programs that demonstrate
the use of conditional statements.

## Topics Covered
- if statement
- if-else statement
- else-if ladder
- Nested conditions

## Programs Included
- IfElseDemo
- EvenOdd
- LargestOfThree
- LeapYear
- GradeCalculator

Language: Java  
Level: Beginner
